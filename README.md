# 🧠 Socket Server in C++

This is a simple socket server framework in C++ that supports both **IPv4** and **IPv6**, designed with modular classes: `Socket`, `Binder`, and `Listener`.

## 🚀 Features

- Base `Socket` class handles both IPv4 and IPv6 sockets.
- `Binder` class abstracts the bind logic for server-side sockets.
- `Listener` adds listening and connection queuing capability.
- Designed with extendability and clarity in mind.
- Written in modern C++17.

## 🏗️ Build Instructions

### Prerequisites
- C++17-compatible compiler (e.g., g++, clang++)
- CMake 3.10+
